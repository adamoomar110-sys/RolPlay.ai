# RolPlay.ai

RolPlay.ai is an interactive role-playing platform powered by AI.

## Features
- **Dynamic Scenarios**: Experience diverse role-playing environments.
- **AI-Driven NPC**: Interact with intelligent characters.
- **Session History**: Track and save your role-playing progress.

## Setup
1. Activate virtual environment:
   ```bash
   .\venv\Scripts\activate
   ```
2. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
